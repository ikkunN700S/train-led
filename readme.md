# 行き先表示機シミュレータ
v1.9.1 (2026年05月27日リリース) 準拠

## 1. 本プロジェクト概要
本作「行き先表示機シミュレータ」はJR東海様の315系の行き先表示機を再現すべく、私個人が自己満足のために制作しているものです。仕様変更等は予告なく実施する可能性があります。

## 2. 使用方法
### 2-1. 在来線版
[https://ikkunn700s.github.io/train-led/](https://ikkunn700s.github.io/train-led/)にアクセスしてください。

表示の変更にはその下にある制御部分を操作してください。変更できるのは以下の要素です。

- 号車番号 (空欄で非表示)
- 種別
- 行き先
- 行き先 (ローマ字)

表示は入力の度に更新されます。

v1.9.0より種別のカラー、テキストが変更可能になりました。  
「**種別・カラー設定**」から各種別のカスタマイズが可能です。プリセットを読み込んで一括設定することもできます。変更後は「適用」ボタンを押してください。

また現在の表示内容を画像としてダウンロードすることができます。ダウンロードしたい内容が表示されているタイミングで「画像として保存ボタン」を押してください。

### 2-2. 新幹線版
本作には新幹線版もあります。
画面下の「新幹線の行き先表示機シミュレータへ」ボタンまたは[https://ikkunn700s.github.io/train-led/shinkansen.html](https://ikkunn700s.github.io/train-led/shinkansen.html)にアクセスしてください。

表示の変更にはその下にある制御部分を操作してください。変更できるのは以下の要素です。。

- 種別
- 号数(こだま〇〇号の〇〇部、空欄で非表示)
- 行き先
- 行き先(ローマ字)

表示は入力の度に更新されます。

また現在の表示内容を画像としてダウンロードすることができます。ダウンロードしたい内容が表示されているタイミングで「画像として保存ボタン」を押してください。
